Plantconvert User Guide
#######################

.. topic:: Summary

    :Version: |version|
    :Release: |release|
    :Date: |today|


You can use the plantconvert package to load your plant file and convert it into another format.

So far, the formats available are [`.mtg`](https://mtg.readthedocs.io/en/latest/user/intro.html), [`.opf`](https://amapstudio.cirad.fr/private/centeredopfmanagement), [`.gltf`](https://www.khronos.org/gltf/) and [`.vtk`](http://www.princeton.edu/~efeibush/viscourse/vtk.pdf)

You can check the tutorials to get the packages possibilities.
